# getting started

Install deno if you haven't already:

```sh
curl -fsSL https://deno.land/x/install/install.sh | sh
```

Clone the repo:

```sh
git clone git@github.com:Jabolol/blog.git .
```

Run the development server:

```sh
deno task dev
```
